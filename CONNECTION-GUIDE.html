🐍 Part 1: Setup Python Backend
1
Create Backend Folder & Add Files
Setup your Python project structure
TERMINAL / POWERSHELL
# Go to your main project folder
cd D:\Fashion-Recommendation-System

# Create backend folder
mkdir backend
cd backend
📋
Copy app.py and requirements.txt files into this backend/ folder. Also copy your embeddings.pkl and filenames.pkl files here from Google Colab.
2
Create Python Virtual Environment
Isolate your Python packages — best practice!
TERMINAL (inside backend/ folder)
# Create virtual environment named "venv"
python -m venv venv

# Activate it on Windows
venv\Scripts\activate

# You will see (venv) appear at start of terminal line
# like: (venv) PS D:\Fashion-Recommendation-System\backend>
✅
When you see (venv) at the start of your terminal line, the virtual environment is active!
3
Install Python Packages
Install Flask, TensorFlow, and all required libraries
TERMINAL (venv must be active)
# Install all packages from requirements.txt
pip install -r requirements.txt

# This will take 5-10 minutes (TensorFlow is large)
# Wait until you see: Successfully installed...
⏳
TensorFlow is a large package (~500MB). Be patient — it will take a few minutes to download and install.
4
Download Dataset Images from Google Colab
Get the embeddings and images from your Colab notebook
💡
In Google Colab, run these cells to download files:
GOOGLE COLAB — Add this cell
# Download your saved files from Colab to your computer
from google.colab import files

# Download embeddings (image fingerprints)
files.download('embeddings.pkl')

# Download filenames list
files.download('filenames.pkl')
📁
Save embeddings.pkl and filenames.pkl inside your backend/ folder. Also copy the fashion-product-images-small/images/ folder into backend/.
5
Start Flask Backend Server
Run your Python API!
TERMINAL (inside backend/ folder, venv active)
# Start Flask server
python app.py

# You should see:
⏳ Loading ResNet50 model...
✅ Model loaded!
⏳ Loading saved embeddings...
✅ Loaded 5000 image embeddings!
✅ NearestNeighbors ready!
🚀 Fashion Recommendation API running!
📡 URL: http://localhost:5000
🎉
Test it! Open browser and go to http://localhost:5000/health — you should see JSON response with status "running"!
⚛️ Part 2: Setup React Frontend
6
Open New Terminal for React
Keep Flask terminal open — open a new separate terminal
⚠️
IMPORTANT: Keep your Flask terminal running! Open a completely new terminal window for React.

In VS Code: Click + button in terminal panel to open new terminal
NEW TERMINAL
# Go to frontend folder
cd D:\Fashion-Recommendation-System\frontend

# Install packages (if not done already)
npm install
7
Replace App.jsx with New Code
Use the new App.jsx that connects to Flask
📋
1. Open frontend/src/App.jsx in VS Code
2. Press Ctrl + A → select all
3. Press Delete
4. Paste the new App.jsx code (from the file I created)
5. Save with Ctrl + S
8
Start React App
Launch the frontend development server
FRONTEND TERMINAL
# Start React development server
npm run dev

# You should see:
  VITE v5.x  ready in 300ms
  ➜  Local:   http://localhost:5173/
🎉
Open http://localhost:5173 in your browser. You should see a green banner saying "✅ AI Backend Connected"!

🐛 Common Problems & Fixes
ProblemFix
Red banner "Flask backend is offline"Run python app.py in backend folder
CORS error in browser consoleMake sure flask-cors is installed and CORS(app) is in app.py
embeddings.pkl not foundRun the Colab notebook first, then download pkl files to backend/
Images not showingMake sure fashion images folder is inside backend/
TensorFlow install errorTry pip install tensorflow-cpu (lighter version)
Port 5000 already in useChange port in app.py: app.run(port=5001) and update API_URL in App.jsx
🏆
Remember: You need TWO terminals running at the same time!
Terminal 1 → python app.py (Flask on port 5000)
Terminal 2 → npm run dev (React on port 5173)
